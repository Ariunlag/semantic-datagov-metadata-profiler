# Expected Case Study Output

The harassment/bullying fixture should produce a metadata report showing:

- a DCAT dataset with public access level, publisher, contact point, keywords, and one CSV distribution
- terminology around harassment, bullying, school safety, education, and public safety
- a distribution table with CSV and text/csv values
- an opt-in resource schema profile with incident id, report date, school name, city, status, category, count, percent, latitude, and longitude fields

The key research point is that catalog metadata identifies the topic and resource pointer, while resource-level profiling reveals the actual analytic schema and field roles.
